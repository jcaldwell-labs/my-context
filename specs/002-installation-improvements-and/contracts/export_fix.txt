### Export with Custom Path

**Input**: `my-context export "Phase 1" --to contexts/phase-1-summary.md`

**Actions**:
1. Create parent directories if needed (`mkdir -p contexts/`)
2. Check if file exists at target path
3. If exists: Display prompt and wait for user input
   ```
   File exists: contexts/phase-1-summary.md
   Overwrite? (y/N): _
   ```
4. If user enters 'y' or 'Y': Overwrite file and continue
5. If user enters anything else or just Enter: Cancel export
6. Write markdown to specified path (if not cancelled)
7. Print success message

**Output (success)**:
```
Exported context "Phase 1" to contexts/phase-1-summary.md
```

**Output (cancelled)**:
```
Export cancelled.
```

**Exit Code**: 0 (success), 2 (user cancelled overwrite)
